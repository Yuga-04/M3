## **Open System Task Analysis (OSTA)**

**Definition:**
Open System Task Analysis (OSTA)** is a **systematic method used in Humanâ€“Computer Interaction (HCI)** to analyze and understand how users perform tasks in a *real-world environment* (open system), where external factors influence user performance.
